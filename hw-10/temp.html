Summary: Latitude vs. Maximum Temperature



The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analaysi, we first pulled data from the OpenWeatherMap API to asseble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at incldued: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as the explaintion and discriptions of any trends and correleations witnessed.
