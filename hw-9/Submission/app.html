This homework uses Python, Flask and SQL to analyze data related to weather pattersn in Oahu.

You can use this app to figure out if the weather in Oahu is going to be sunny or rainy during your next vacation. There are three avaible routes for this API that are listed below.

Route 1: Precipitation

If you want to find out the chances that it will be raining when you go to Oahu, you can use the following route: /api/v1.0/precipitation

Route 3: List of All Weather Stations

If you want to get a list of all of the weather stations that we have data for, you can use the following route: /api/v1.0/stations

Route 2: Temperature

If you want to know the weather when you go to Oahu, you can use the following route: /api/v1.0/tobs

Route 4: Historial Temperature

If you want to get a historial list of the temperatures on Oahu for the past year, you can use the following route: /api/v1.0/<start>
